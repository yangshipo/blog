# 个人合作指南

基于我们聊的所有内容，为你量身整理。

---

## 第一部分：认识自己的模式

在合作中，你要先知道自己的"自动反应"：

### 你的触发点
| 触发情况 | 你的自动反应 | 真正的原因 |
|---------|------------|-----------|
| 感觉利益被动了 | 紧张、想控制 | 曾经被信任的人出卖 |
| 感觉不被重视 | 愤怒、评判对方 | 害怕被忽略意味着被抛弃 |
| 感觉付出不公平 | 不满、想对抗 | 努力过却没被认可的伤痛 |
| 感觉事情不在掌控中 | 焦虑、想全部自己做 | 不信任别人能做好 |

> **下次出现这些反应时，先停10秒，问自己：**
> 
> **"这是现在这个人在伤害我，还是我的旧伤在疼？"**

## 第二部分：选人策略

### 不再凭"感觉"信任，而是凭"观察"

**观察周期：至少1-3个月**

观察什么：

| 观察项 | 怎么看 |
|-------|--------|
| 他对别人怎么样 | 不只是对你，看他怎么对对他没用的人 |
| 他说到做到吗 | 不看大话，看小事上是否兑现承诺 |
| 利益冲突时的反应 | 他是只顾自己还是愿意协商 |
| 背后怎么说别人 | 当面一套背后一套的人远离 |
| 有没有主动帮过人 | 不图回报的小举动最能看出人品 |

### 把人分三层

```
第一层：核心合作者（1-2人）
  - 观察过很久
  - 利益方向一致
  - 在小事上证明过靠谱
  - 可以互相交底

第二层：普通合作者
  - 正常工作配合
  - 合作但不交底
  - 保持职业化的关系

第三层：远离的人
  - 明确不可信的
  - 损害过你或别人的
  - 保持距离，保护自己
```

> **不是所有人都值得你同等对待，但也不是所有人都要防着。**

## 第三部分：合作中的具体原则

### 原则一：利益绑定优先于个人信任

> 不找"好人"，找**利益一致的人**。

- 同一个项目，做成了大家都受益
- 互相有对方需要的资源
- 合作比竞争对双方都划算

**利益一致的时候，人是最可靠的。**

### 原则二：永远保护自己的底线

> 上次的教训不是"不该信任"，而是"把所有希望押在一个人身上"。

以后做到：
- 重要的成果**留痕迹**，邮件、文档、记录
- 关键的事**让多个人知道**，不只依赖一个上级
- 自己的功劳**自己也要说出来**，不等别人替你说
- 口头承诺**不算数**，看行动

### 原则三：先从小事开始交付信任

不要一上来就全盘信任，也不要完全不信任：

```
第一步：交出一件小事（影响不大的）
    ↓
观察对方怎么处理
    ↓
做得好 → 下次交一件稍大的事
做不好 → 退回普通合作，不翻脸
    ↓
多次验证后 → 逐步升级信任层级
```

> **信任是一点一点给的，不是一次性的。**

### 原则四：允许别人不完美

这是你最难做到的，但最重要的：

| 以前的标准 | 新的标准 |
|-----------|---------|
| 一次让我失望就淘汰 | 看整体，偶尔失望是正常的 |
| 利益冲突时必须选我 | 不主动害我就及格 |
| 100%站我这边 | 大多数时候靠谱就够了 |

> **完美的人不存在，够好的人值得留下。**

### 原则五：学会表达而不是忍或爆

以前你可能是：

> 不满 → 忍着 → 积累 → 爆发或切断关系

以后可以试试：

> 不满 → **及时、平静地说出来**

话术参考：
- ❌ "你怎么能这样？"
- ✅ **"这件事让我觉得不太公平，我们可以聊聊吗？"**
- ❌ "算了，靠你不如靠自己"
- ✅ **"这件事对我很重要，我希望你能重视"**

## 第四部分：给自己的提醒

写下来，放在手机里，时不时看看：

> **1. 那个人出卖了我，但不是所有人都会。**
> 
> **2. 保护自己是对的，但把所有人推开是在惩罚自己。**
> 
> **3. 我不需要100%信任一个人，70%就够了。**
> 
> **4. 靠自己很强，但有人一起走会更远。**
> 
> **5. 允许别人不完美，也允许自己不完美。**
> 
> **6. 信任是慢慢给的，不是一次赌博。**
> 
> **7. 我值得被好好对待，所以我不关上这扇门。**

## 第五部分：从今天开始

不需要做很多，就做三件事：

### 这周
> 观察你身边的同事，找一个你觉得"还算靠谱"的人，不做任何事，只是**注意他**。

### 这个月
> 找一件小事，跟这个人合作，**看看结果**。

### 三个月后
> 回头看看，你的世界有没有松动一点点。

## 最后一句话

你在阿里活了7年，被背叛过，一个人扛到现在。

**你已经证明了你足够强。**

现在不是要你变弱，而是——

> **强大的人敢一个人走，**
> 
> **更强大的人敢再次让别人走进来。**

**你可以的。慢慢来。**
