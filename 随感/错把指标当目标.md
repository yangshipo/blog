# 错把指标当目标


## 1. 一条完整的链：动作、指标、结果、目标

```
动作  →  指标  →  结果  →  目标
加班    代码行数   价值     好的生活/事业
```

- **目标**：你真正想要的（活得好、做成事）
- **结果**：达成目标的真实成效（创造的价值、解决的问题）
- **指标**：用来*测量*结果的代理物（KPI、数字、可见信号）
- **动作**：你具体干的事



> **错把"用来测量目标的指标"，当成了"目标本身"。**

你简化成"错把指标当目标"，完全抓住了。

### 为什么"指标"这个词更狠、更准

因为它直接点破了**"骗人发生在哪一环"**。

不是"结果"在骗你——你追求结果是对的，结果是好东西。
是**"指标"在骗你**——指标天然有缺陷，它总会漏掉一些东西，或者扭曲一些东西。

这甚至有个专门的定律，叫 **古德哈特定律（Goodhart's Law）**：

> **"当一个指标变成了目标，它就不再是个好指标了。"**

意思是：指标本来是个还算靠谱的"温度计"，能反映真实情况。但一旦你开始为了让温度计读数好看而行动，大家就都去**操纵温度计**，而不是真的去调温度了。于是这个指标就失真、报废了。

回到你昨天那个例子——"加班、提交代码"就是个**指标**。它本来想测量的真实结果是"你创造了价值"。但一旦公司**只看这个指标**，所有人就开始表演加班、灌水代码，于是这个指标彻底失真，再也测不出谁真在创造价值了。这就是古德哈特定律活生生的演示。

### 所以改完之后，整套话术都更利索了

- ❌ 错把结果当目标 →（有点别扭，因为结果其实是好东西）
- ✅ **错把指标当目标** →（精准，因为指标才是那个骗人的代理物）

而且你这一改，连带把行动指南也变清楚了：

> **盯着结果，利用指标，但永远别效忠于指标。**

- **盯着结果**——你真正要的是创造价值、过好生活，这个不能丢。
- **利用指标**——指标能帮你和别人快速判断进展，是个有用工具，别浪费。
- **别效忠指标**——当指标和真实结果打架时（比如为了压通话时长草草挂客户电话），毫不犹豫站结果那边。



## 2. 动作、指标、结果、目标 与 OKR

链条：

```
动作  →  指标  →  结果  →  目标
                  ↑
              KR 站在这
```

KR 用"结果"这个词，是在告诉你：

> **它要描述的是"目标达成后世界变成什么样"（结果），而不是"你做了多少动作"（指标常退化成的东西）。**

举个对比就清楚了：

- ❌ "本季度发 20 篇推广文" —— 这是**动作/产出量**，是最差的 KR，根本不配叫 Result
- ✅ "新用户7日留存率从 30% 到 50%" —— 这才是 **Result**，描述的是世界发生的真实变化

所以 R=Result 这个用词，本身就是在**抵抗"错把指标/动作当目标"**。你纠正得越较真，越能看出设计者用心。

### 那"指标"在这套体系里到底站哪？

咱们得给指标重新安排个准确的位置，不能再让它冒充 KR 了：

```
O      目标       让新用户爱上产品（定性、方向）
 │
KR     关键结果    留存率达到 50% / NPS 达到 Y（"成功长什么样"）
 │
指标    Metric     留存率、NPS、使用次数……（测量 KR 的尺子）
 │
动作    Action     做 onboarding 优化、改新手引导……（你具体干的）
```

看出区别了吗——

- **KR（结果）**说的是 **"要到达哪个状态"**（留存率*达到* 50%）
- **指标（Metric）**是那把 **"量你到没到"的尺子**（留存率这个量本身）

它俩长得像，因为 KR 经常**借一个指标的数值来表达**自己（"留存率到50%"）。但本质不同：**KR 是你想要的那个真实状态，指标只是测它的工具。** 这恰恰又回到咱们第一轮那个核心区分了。

## 所以三轮下来，这条链现在是完整的、自洽的

| 概念 | 在 OKR 里 | 角色 | 会不会骗人 |
|---|---|---|---|
| 目标 | O | 真正想要的方向 | 不会，但容易空 |
| 结果 | KR | 目标达成时的真实状态 | 不会，是好东西 |
| 指标 | （KR借用的数值） | 测量结果的尺子 | **会，古德哈特陷阱在此** |
| 动作 | 待办/项目 | 具体干的事 | 容易被误当成结果 |

**两个最常见的错位，正好是两端：**

1. **错把指标当目标**（古德哈特）—— 为了让尺子读数好看而行动
2. **错把动作当结果**（写出"发20篇文"这种假KR）—— 把"忙"当成"成"

你昨天那个加班的憋屈，其实**两种错位都占了**：加班是"动作"，被当成了"结果"；而老板看的提交量是"指标"，被当成了"目标"。难怪憋屈——你陷在了 O 和 Action 之间，被两层假象夹着。
